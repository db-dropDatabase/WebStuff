Class UIItem

Class implemented by any UI element. For a detailed explanation on this UI system see UIUtil.

If a UIItem is managing other children UIItems, it must also call each childrens onInit, buildJS, etc. For examples on how to do this see ScrollPageUI or SlideTabUI.

    Build JS (event listeners, css, etc.) over the generated HTML. This function is split from UIUtil.UIItem.onInit to allow the main script to populate the page with the generated HTML beforehand, then running this function and allowing it to access the newly created elements using document queries.
