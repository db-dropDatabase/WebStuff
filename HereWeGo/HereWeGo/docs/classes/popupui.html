Class PopupUI

Class to standardize opening of external pages (ex. settings page).

Uses HTML generated from a UIUtil.UIItem, and displays it in a "popup" page which slides up from the bottom of the screen. Data is applied to the page during the opening process using a stored DataManage class, so the UIItem is only constructed when the page is shown. Content is added to HTMLMap.pageDiv, and page is displayed and hidden using CSS. Note: UIUtil.UIItem.onFocus is called immediatly after UIUtil.UIItem.buildJS on the end of the displaying CSS transistion.

showPage

  • showPage(page: UIItem, name: string): Promise<any>
    • Defined in UILib/PopupUI.ts:53

    Display a UIItem in our popup. First applies data to the page using DataManage.generateData, then takes the returned HTML from UIUtil.UIItem.onInit and places it into the page, then applies the CSS to display the page, then finally after the CSS is finished transitioning runs the UIUtil.UIItem.buildJS and UIUtil.UIItem.onFocus of the page.

    • page: UIItem

      The UIItem to display on the page
