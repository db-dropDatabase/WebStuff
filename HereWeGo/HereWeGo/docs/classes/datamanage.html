Class DataManage

There are a few stages required to making any sort of data from the interet useful to a user: fetch, store, parse, route, and display. This class controls all of those stages accept display, which is handled by the UIUtil system, and fetch, which is handled by the GetLib system.

Data is fetched from a Google Apps Script running here, which controls all data except images. This cloud script is hosted through the wilsoncomputerscience@gmail.com google account. Data from this script has been absctracted into components (such as calendar, images), and each component has been written it's own class to parse, store, and return the data. This class simply fetches data from the internet, and runs the individual class code on it. These classes implement the interface DataInterface.

For peace of mind, this class also ensures the database is initialized and built using DBManage. A pointer to the database is passed to the class during contruction.
