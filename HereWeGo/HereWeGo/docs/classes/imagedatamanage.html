Class ImageDataManage

Class to parse and store background images from the cloud.

Images URLs are determined from the cloud google apps script, this class simply downloads them and catalogs them for retrieval when the app starts. Stores images in the local cached using the cordova file plugin and retreives them using a file path stored in the database.

Pretty complicated due to the synchronization required to manage the images database entries.
